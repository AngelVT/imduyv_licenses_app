Folios de conteo

Antes de comenzar a utilizar el sistema, es necesario determinar los folios iniciales para cada uno de los 15 formatos de registros que se manejan. Sigue estos pasos:

  • Identifica el último folio de cada tipo que se haya generado fuera del sistema.
  • Por ejemplo: si el último folio del tipo C es 150, este será el valor que deberás indicar.
  • Ingresa estos folios en su campo correspondiente. El sistema comenzará a generar automáticamente los folios a partir del siguiente número (en este ejemplo, el próximo sería 151).

Importante:

  • Esta configuración solo se realiza una vez y no se puede modificar posteriormente.
  • Asegúrate de ingresar los valores correctamente antes de confirmar.

Si tienes dudas, consulta antes de proceder.
